# 微信支付快速参考

## 🚀 快速开始

### 1. 配置渠道（数据库）

```sql
-- 配置 JSAPI/H5/APP 支付（配置相同）
INSERT INTO channel_configs (app_id, channel, config, status)
VALUES ('your_app_id', 'wechat_jsapi', '{"mch_id":"1900000109","serial_no":"xxx","api_v3_key":"xxx","private_key_path":"/path/to/key.pem"}', 'active');
```

### 2. 调用支付接口

```bash
# JSAPI 支付
curl -X POST http://localhost:8080/api/v1/checkout -H "Content-Type: application/json" -d '{
  "app_id": "your_app_id",
  "out_trade_no": "ORDER001",
  "amount": 100,
  "subject": "测试商品",
  "channel": "wechat_jsapi",
  "extra_data": {"app_id": "wx123", "openid": "oUpF8uMuAJO"}
}'

# H5 支付
curl -X POST http://localhost:8080/api/v1/checkout -H "Content-Type: application/json" -d '{
  "app_id": "your_app_id",
  "out_trade_no": "ORDER002",
  "amount": 200,
  "subject": "测试商品",
  "channel": "wechat_h5",
  "extra_data": {"app_id": "wx123", "client_ip": "1.2.3.4", "h5_type": "Wap"}
}'

# APP 支付
curl -X POST http://localhost:8080/api/v1/checkout -H "Content-Type: application/json" -d '{
  "app_id": "your_app_id",
  "out_trade_no": "ORDER003",
  "amount": 300,
  "subject": "测试商品",
  "channel": "wechat_app",
  "extra_data": {"app_id": "wx123"}
}'
```

## 📋 支付方式对比

| 特性 | Native | JSAPI | H5 | APP |
|-----|--------|-------|----|----|
| **场景** | PC扫码 | 公众号/小程序 | 手机浏览器 | 原生APP |
| **必需参数** | app_id | app_id, openid | app_id, client_ip | app_id |
| **返回内容** | code_url | prepay_id + 调起参数 | h5_url | prepay_id + 调起参数 |
| **前端处理** | 生成二维码 | wx.chooseWXPay() | 跳转链接 | WXApi.sendReq() |

## 🔑 ExtraData 参数

### JSAPI 支付
```json
{
  "app_id": "wx1234567890abcdef",    // 必需：微信应用ID
  "openid": "oUpF8uMuAJO_M2pxb1Q9"   // 必需：用户openid
}
```

### H5 支付
```json
{
  "app_id": "wx1234567890abcdef",    // 必需：微信应用ID
  "client_ip": "123.12.12.123",      // 可选：用户IP（默认127.0.0.1）
  "h5_type": "Wap"                   // 可选：Wap/iOS/Android（默认Wap）
}
```

### APP 支付
```json
{
  "app_id": "wx1234567890abcdef",    // 必需：微信应用ID
  "device_id": "device_123456"       // 可选：设备ID
}
```

## 📱 前端集成

### JSAPI（公众号）
```javascript
wx.chooseWXPay({
  timestamp: payInfo.timestamp,
  nonceStr: payInfo.nonce_str,
  package: payInfo.package,
  signType: payInfo.sign_type,
  paySign: payInfo.pay_sign,
  success: (res) => console.log('成功', res)
});
```

### JSAPI（小程序）
```javascript
wx.requestPayment({
  timeStamp: payInfo.timestamp,
  nonceStr: payInfo.nonce_str,
  package: payInfo.package,
  signType: payInfo.sign_type,
  paySign: payInfo.pay_sign,
  success: (res) => console.log('成功', res)
});
```

### H5
```javascript
window.location.href = response.data.pay_url;
```

### APP (iOS)
```objc
PayReq *req = [[PayReq alloc] init];
req.partnerId = payInfo[@"partner_id"];
req.prepayId = payInfo[@"prepay_id"];
req.package = payInfo[@"package"];
req.nonceStr = payInfo[@"nonce_str"];
req.timeStamp = [payInfo[@"timestamp"] intValue];
req.sign = payInfo[@"sign"];
[WXApi sendReq:req completion:nil];
```

### APP (Android)
```java
PayReq req = new PayReq();
req.appId = payInfo.get("app_id");
req.partnerId = payInfo.get("partner_id");
req.prepayId = payInfo.get("prepay_id");
req.packageValue = payInfo.get("package");
req.nonceStr = payInfo.get("nonce_str");
req.timeStamp = payInfo.get("timestamp");
req.sign = payInfo.get("sign");
api.sendReq(req);
```

## ⚠️ 常见错误

| 错误 | 原因 | 解决方案 |
|-----|------|---------|
| "openid is required" | JSAPI缺少openid | 在extra_data中添加openid |
| "app_id is required" | 缺少微信应用ID | 在extra_data中添加app_id |
| "商家参数格式有误" | H5参数错误 | 检查client_ip和h5_type |
| 签名验证失败 | 证书配置错误 | 检查private_key_path |
