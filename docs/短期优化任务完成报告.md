## 2️⃣ 压力测试

### 完成内容

**创建压力测试套件**
- `tests/load/load_test.go` - 完整的压力测试框架
- `tests/load/README.md` - 详细的压力测试指南

**测试场景**

1. **下单接口压力测试** (`TestLoadCheckout`)
   - 目标 QPS: 1,000
   - 并发数: 100
   - 持续时间: 30 秒
   - 性能指标: P95 < 100ms, 错误率 < 1%

2. **查询接口压力测试** (`TestLoadQuery`)
   - 目标 QPS: 5,000
   - 并发数: 200
   - 持续时间: 30 秒
   - 性能指标: P95 < 50ms

3. **10k+ QPS 极限测试** (`TestLoad10kQPS`)
   - 目标 QPS: 10,000
   - 并发数: 500
   - 持续时间: 60 秒
   - 性能指标: 实际 QPS ≥ 10,000, 错误率 < 0.1%

**测试功能**
- ✅ 支持自定义并发数、持续时间、目标 QPS
- ✅ 支持预热期（Ramp-up）逐步增加负载
- ✅ 自动签名和认证
- ✅ 详细的性能统计（平均、最小、最大、P50、P95、P99）
- ✅ 实时 QPS 和错误率计算
- ✅ 支持不同请求类型（下单、查询）

**使用方法**
```bash
# 运行所有压力测试
cd tests/load
go test -v -timeout 10m

# 运行 10k QPS 测试
go test -v -run TestLoad10kQPS -timeout 5m

# 跳过长时间测试
go test -v -short
```
