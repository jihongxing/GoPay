# 多渠道支付快速参考

## 🚀 支持的支付渠道

### 微信支付（4种）
- ✅ `wechat_native` - PC 扫码支付
- ✅ `wechat_jsapi` - 公众号/小程序支付
- ✅ `wechat_h5` - 手机网页支付
- ✅ `wechat_app` - 原生 APP 支付

### 支付宝（4种）
- ✅ `alipay_qr` - PC 扫码支付
- ✅ `alipay_wap` - 手机网站支付
- ✅ `alipay_app` - 原生 APP 支付
- ✅ `alipay_face` - 当面付（线下扫码）

**总计**: 8 种支付方式，覆盖全场景

## 📋 渠道对比

### 场景对比

| 场景 | 微信 | 支付宝 |
|-----|------|--------|
| **PC 扫码** | wechat_native | alipay_qr |
| **手机网页** | wechat_h5 | alipay_wap |
| **APP 支付** | wechat_app | alipay_app |
| **公众号/小程序** | wechat_jsapi | - |
| **线下扫码** | - | alipay_face |

### 配置对比

| 项目 | 微信 | 支付宝 |
|-----|------|--------|
| **商户标识** | mch_id | app_id |
| **签名方式** | RSA + 证书 | RSA |
| **证书管理** | 商户证书 + 平台证书 | 应用私钥 + 支付宝公钥 |
| **金额单位** | 分 | 元（API）/ 分（存储） |
| **配置复杂度** | 高 | 中 |

### 参数对比

| 参数 | 微信 | 支付宝 |
|-----|------|--------|
| **订单号** | out_trade_no | out_trade_no |
| **金额** | amount（分） | total_amount（元） |
| **商品标题** | description | subject |
| **回调 URL** | notify_url | notify_url |

## 📱 API 调用示例


## 🎯 渠道选择指南

### PC 网站

**推荐**: `wechat_native` 或 `alipay_qr`

```javascript
// 根据用户选择的支付方式
const channel = userChoice === 'wechat' ? 'wechat_native' : 'alipay_qr';

// 调用下单接口
const response = await createOrder({
  channel: channel,
  amount: 100,
  subject: '商品名称'
});

// 生成二维码
QRCode.toCanvas(canvas, response.data.qr_code);
```

### 手机网页

**推荐**: `wechat_h5` 或 `alipay_wap`

```javascript
// 检测用户环境
const isWechat = /MicroMessenger/i.test(navigator.userAgent);
const channel = isWechat ? 'wechat_h5' : 'alipay_wap';

// 调用下单接口
const response = await createOrder({
  channel: channel,
  amount: 100,
  subject: '商品名称'
});

// 跳转支付
window.location.href = response.data.pay_url;
```

### 微信内（公众号/小程序）

**推荐**: `wechat_jsapi`

```javascript
// 获取用户 openid
const openid = await getWechatOpenId();

// 调用下单接口
const response = await createOrder({
  channel: 'wechat_jsapi',
  amount: 100,
  subject: '商品名称',
  extra_data: {
    app_id: 'wx1234567890abcdef',
    openid: openid
  }
});

// 调起支付
wx.chooseWXPay({
  timestamp: response.data.pay_info.timestamp,
  nonceStr: response.data.pay_info.nonce_str,
  package: response.data.pay_info.package,
  signType: response.data.pay_info.sign_type,
  paySign: response.data.pay_info.pay_sign,
  success: (res) => console.log('支付成功', res)
});
```

### 原生 APP

**推荐**: `wechat_app` 或 `alipay_app`

```javascript
// 根据用户选择的支付方式
const channel = userChoice === 'wechat' ? 'wechat_app' : 'alipay_app';

// 调用下单接口
const response = await createOrder({
  channel: channel,
  amount: 100,
  subject: '商品名称'
});

// 调起支付（iOS/Android 原生代码）
if (channel === 'wechat_app') {
  // 调起微信支付
  WXApi.sendReq(response.data.pay_info);
} else {
  // 调起支付宝
  AlipaySDK.pay(response.data.pay_info.order_string);
}
```

### 线下收银

**推荐**: `alipay_face`

```javascript
// 扫描用户付款码
const authCode = scanPaymentCode();

// 调用下单接口
const response = await createOrder({
  channel: 'alipay_face',
  amount: 100,
  subject: '商品名称',
  extra_data: {
    auth_code: authCode
  }
});

// 当面付是同步支付，直接显示结果
if (response.data.pay_info.trade_status === 'TRADE_SUCCESS') {
  console.log('支付成功');
}
```

## ⚠️ 常见错误

| 错误信息 | 可能原因 | 解决方案 |
|---------|---------|---------|
| "openid is required" | JSAPI 缺少 openid | 在 extra_data 中添加 openid |
| "auth_code is required" | Face 缺少付款码 | 在 extra_data 中添加 auth_code |
| "签名验证失败" | 证书/密钥配置错误 | 检查配置是否正确 |
| "金额格式错误" | 金额单位错误 | 统一使用分作为单位 |
