# GoPay 管理后台启动指南

## 快速启动

### 1. 启动数据库

```bash
# 使用 Docker Compose
docker-compose up -d postgres

# 或者使用已有的 PostgreSQL
# 确保数据库已创建: gopay
```

### 2. 配置环境变量

创建 `.env` 文件:

```bash
# 数据库配置
DB_HOST=localhost
DB_PORT=5432
DB_USER=gopay
DB_PASSWORD=your_password
DB_NAME=gopay

# 服务配置
SERVER_PORT=8080
SERVER_ENV=development

# 日志配置
LOG_LEVEL=info
LOG_FILE=logs/gopay.log
```

### 3. 启动服务

```bash
# 方式 1: 直接运行
go run cmd/gopay/main.go

# 方式 2: 构建后运行
go build -o gopay cmd/gopay/main.go
./gopay

# 方式 3: 使用 Air 热重载（推荐开发环境）
air
```

### 4. 访问管理后台

打开浏览器访问: http://localhost:8080/admin

## 故障排查

### 问题 1: 页面显示 500 错误

**原因**: 模板文件未找到

**解决方案**:
```bash
# 检查模板文件是否存在
ls -la web/admin/templates/

# 确保在项目根目录运行
pwd  # 应该显示 .../GoPay
```

### 问题 2: 静态资源 404

**原因**: 静态文件路径不正确

**解决方案**:
```bash
# 检查静态文件是否存在
ls -la web/admin/static/

# 确保目录结构正确
web/admin/
├── static/
│   ├── css/
│   └── js/
└── templates/
```

### 问题 3: API 返回错误

**原因**: 数据库连接失败

**解决方案**:
```bash
# 检查数据库连接
psql -h localhost -U gopay -d gopay -c "SELECT 1"

# 检查环境变量
echo $DB_HOST
echo $DB_PORT
```

### 问题 4: 图表不显示

**原因**: Chart.js 加载失败

**解决方案**:
- 检查网络连接（Chart.js 从 CDN 加载）
- 打开浏览器开发者工具查看错误
- 检查是否被防火墙拦截

## 生产部署

### 1. 构建

```bash
# 构建二进制文件
go build -ldflags="-s -w" -o gopay cmd/gopay/main.go

# 压缩（可选）
upx gopay
```

### 2. 配置 Systemd

创建 `/etc/systemd/system/gopay.service`:

```ini
[Unit]
Description=GoPay Payment Gateway
After=network.target postgresql.service

[Service]
Type=simple
User=gopay
WorkingDirectory=/opt/gopay
ExecStart=/opt/gopay/gopay
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
```

启动服务:
```bash
sudo systemctl enable gopay
sudo systemctl start gopay
sudo systemctl status gopay
```

### 3. 配置 Nginx

```nginx
server {
    listen 80;
    server_name admin.example.com;

    location / {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    # 静态文件缓存
    location /admin/static/ {
        proxy_pass http://localhost:8080;
        expires 7d;
        add_header Cache-Control "public, immutable";
    }
}
```

## 安全加固

### 1. 添加认证

// 简单的 Basic Auth
authorized := router.Group("/admin", gin.BasicAuth(gin.Accounts{
    "admin": "password",
}))

### 2. IP 白名单

```go
func IPWhitelist(allowedIPs []string) gin.HandlerFunc {
    return func(c *gin.Context) {
        clientIP := c.ClientIP()
        for _, ip := range allowedIPs {
            if ip == clientIP {
                c.Next()
                return
            }
        }
        c.AbortWithStatus(403)
    }
}

router.Use(IPWhitelist([]string{"127.0.0.1", "192.168.1.100"}))
```

## 常见问题

**Q: 如何修改端口？**
A: 修改 `.env` 文件中的 `SERVER_PORT`

**Q: 如何添加新的统计指标？**
A: 修改 `internal/admin/web_handler.go` 中的 `GetStats` 方法

**Q: 如何自定义样式？**
A: 修改 `web/admin/static/css/main.css`

**Q: 如何添加新页面？**
A: 
1. 在 `web/admin/templates/` 创建新模板
2. 在 `web_handler.go` 添加路由和处理函数
3. 在 `layout.html` 添加导航链接
